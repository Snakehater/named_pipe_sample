# Interprocess Communication sample with a fifo/named pipe

## Installation for arch

### Clone this repository
`
git clone git@github.com:Snakehater/named_pipe_sample.git
`

### Install requirements
`
pacman -S cmake ninja
`

## Setup for linux
`
mkdir build;
cd build;
cmake -G ninja ..;
`

## Build
`
ninja main
`

## Run
`
./main
`
